┌──────────────────────────────────────────────────────────────────────────────┐
│                    PROBLEM: Location Not Accurate                            │
└──────────────────────────────────────────────────────────────────────────────┘

    ✓ Move outdoors (better GPS signal)
    ✓ Wait 10-20 seconds (GPS stabilizing)
    ✓ Enable "High Accuracy" mode
    ✓ Refresh page

┌──────────────────────────────────────────────────────────────────────────────┐
│                    PROBLEM: "Location Access Denied"                         │
└──────────────────────────────────────────────────────────────────────────────┘

    You clicked "Block" by mistake!
    
    Fix:
    1. Click lock icon (🔒) in address bar
    2. Find "Location" setting
    3. Change from "Block" to "Allow"
    4. Refresh page (F5)
